# ADF Monte Carlo Simulation

This project simulates the Dickey–Fuller test statistic under a unit root process with drift using Monte Carlo methods.

## Objective

The goal is to study the finite-sample behavior of the Dickey–Fuller t-statistic when the data are generated by a nonstationary process:

$$Y_t = \mu + Y_{t-1} + \epsilon_t$$

where $\epsilon_t \sim N(0,1)$.

The Dickey–Fuller test statistic **does not follow a standard t distribution under the null of a unit root**. Monte Carlo simulation helps approximate its empirical distribution.

## Method

For each simulation:

1. Generate a unit root process with drift  
2. Discard an initial burn-in period  
3. Estimate the regression  

$$\Delta Y_t = \alpha + \pi Y_{t-1} + u_t$$

4. Record the test statistic on $Y_{t-1}$

This process is repeated 10,000 times to approximate the empirical distribution of the Dickey–Fuller statistic.

## Tools

- **Python**  
- **NumPy**  
- **statsmodels**  

## Output

The script reports:

- The **5th percentile** of the simulated test statistic  
- The **mean** of the simulated test statistic
